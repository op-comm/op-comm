# OP-COMM

OP-COMM is a go library for managing websocket connections.

## Installation

```bash
go get github.com/op-comm/op-comm
```

## Development

### Prerequisites (for development)

- Go (1.22+)
- make

#### Build Project

```bash
make build
```

#### Run Tests

```bash
make test
```

#### Format & Cleanup

```bash
make cleanup
```

This should be run before every commit
